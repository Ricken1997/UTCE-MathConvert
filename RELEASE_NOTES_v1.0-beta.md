# UTCE MathConvert Free Beta 1.0 Release Notes

## Release Type

Free Beta

## Highlights

- Added Word .docx export.
- OMML is embedded as native Word equation objects.
- Added basic composite expression support.
- Added operator conversion:
  - * → ×
  - / → ÷
- Improved warning detection.
- Improved GUI layout.

## Recommended Use

- Reports
- Undergraduate theses
- Basic academic documents
- Work documents requiring Word equations

## Known Limitations

Nested and complex expressions are planned for v1.1 and later.
