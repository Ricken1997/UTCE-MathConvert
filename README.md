# UTCE MathConvert Beta 1.0

Based on v20.0-core-product

## Overview

UTCE MathConvert is an experimental plain-math conversion tool for generating LaTeX, MathML, and True OMML output.

## Features

- Greek Symbols
- Fraction
- Square Root
- Superscript
- Subscript
- Subscript + Superscript
- Summation
- Product
- Integral
- Limit
- Matrix
- Cases
- Derivative

## Outputs

- LaTeX
- MathML
- True OMML

## Usage

Run:

```bash
python3 app.py
